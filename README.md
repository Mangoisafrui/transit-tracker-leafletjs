# transit-tracker-leafletjs

A real-time transit tracking web application built with Leaflet.js that displays live vehicle positions on an interactive map.

## Features
- Interactive map powered by Leaflet.js
- Fetches real-time bus data from the Halifax Transit API
- Filters results to show buses on routes 1-10 only
- Converts API data to GeoJSON format for map plotting
- Custom bus icons as map markers
- Rotated markers indicating the direction each bus is travelling
- Clickable popups showing additional vehicle information
- Auto-refreshes map at set intervals without reloading the page

## Technologies
- JavaScript
- Leaflet.js
- GeoJSON
- Halifax Transit API
- HTML
- CSS

## API
- Live bus data: `https://halifax-transit-data.onrender.com/vehicles`

## How to Run
Open `index.html` in your browser — no installation required.
